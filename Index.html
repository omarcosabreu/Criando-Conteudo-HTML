HTML básico - códigos HTML

Definição

É uma linguagem utilizada para desenvolvimento de documentos e páginas web
por meio de marcadores (tags) e atributos.

O conteudo do HTML é indicado por tags do HTML

O que é preciso para criar um a página html?

Para desenvolver códigos HTML para sites é necessário:

  1. um navegador (ex.:Google Chrome)
  2. um simples editor de texto (ex.:bloco de notas) ou outros (ex.:Visual Studio Code)
